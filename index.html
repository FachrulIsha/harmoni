Panduan Penggunaan Website

HARMONI

"Hargai Perbedaan, Momen Kebersamaan, Niatkan Kebaikan"

Website pembelajaran PAI untuk kelas 5 SD yang mengajarkan tentang akhlak mulia: Kejujuran, Tanggung Jawab, serta Saling Menghargai & Menghormati. Mari belajar bersama dalam harmoni!

Langkah 1: Login

✓ Masukkan Nama Lengkap dan Kelas (contoh: 5A)

✓ Klik tombol "Masuk" untuk memulai

✓ Data akan disimpan otomatis di browser kamu

Langkah 2: Halaman Home

✓ Menyambut siswa dengan ucapan selamat datang

✓ Menampilkan fitur-fitur unggulan website

✓ Klik "Panduan Penggunaan" untuk melihat petunjuk ini

Langkah 3: Belajar Materi

✓ Klik menu "Materi" untuk memulai belajar

✓ Pilih salah satu dari 3 materi: Kejujuran, Tanggung Jawab, atau Saling Menghargai & Menghormati

✓ Klik tombol "Baca Materi" untuk membaca penjelasan lengkap

✓ Baca dengan saksama karena materi akan diujikan di kuis

Langkah 4: Mengerjakan Kuis

✓ Klik menu "Kuis" untuk melihat pilihan kuis

✓ Tersedia 4 kuis dengan 10 soal masing-masing:

- Kuis 1: Soal tentang Kejujuran

- Kuis 2: Soal tentang Tanggung Jawab

- Kuis 3: Soal tentang Saling Menghargai & Menghormati

- Kuis 4: Soal gabungan dari semua materi

✓ Pilih kuis yang ingin dikerjakan

✓ Jawab 10 soal pilihan ganda dengan memilih salah satu opsi

✓ Setiap jawaban benar mendapat 10 poin (total 100 poin per kuis)

✓ Klik "Soal Selanjutnya" untuk melanjutkan

✓ Setelah selesai, klik "Selesai" untuk melihat skor

Langkah 5: Bermain Game

✓ Klik menu "Game" untuk melihat pilihan game

✓ Tersedia 3 game seru:

- Game 1: Frog Jump

- Game 2: Spin Wheel

- Game 3: Zep Quiz

✓ Klik salah satu game untuk membuka di tab baru

✓ Ikuti petunjuk permainan di masing-masing game tersebut

Langkah 6: Melihat Profil

✓ Klik menu "Profil" untuk melihat data diri

✓ Lihat total poin yang sudah dikumpulkan

✓ Lihat riwayat nilai kuis yang pernah dikerjakan

✓ Poin akan terus bertambah setiap mengerjakan kuis

✓ Gunakan tombol "Hapus Data" untuk menghapus semua data yang tersimpan

Langkah 7: Melihat Tim Pengembang

✓ Klik menu "Tim Pengembang" untuk melihat informasi tim pembuat website

✓ Terdapat 5 anggota tim pengembang

Tips Belajar

✓ Baca materi terlebih dahulu sebelum mengerjakan kuis

✓ Kerjakan kuis dengan jujur dan teliti

✓ Kumpulkan poin sebanyak-banyaknya

✓ Mainkan game untuk belajar sambil bersenang-senang

✓ Selamat belajar, semoga sukses!
